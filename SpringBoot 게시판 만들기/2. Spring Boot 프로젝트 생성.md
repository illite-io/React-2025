# 2) 스프링 부트 프로젝트 생성

## 2-1. Spring Initializr(웹)에서 생성 (가장 흔한 경로)

* **Project**: Maven
* **Language**: Java
* **Spring Boot**: 3.3.x (안정 버전)
* **Group**: `com.example`
* **Artifact**: `board-api`
* **Java**: 21
* **Dependencies**:

  * Spring Web
  * Validation
  * Lombok
  * MyBatis Framework
  * MySQL Driver
* Generate → zip 다운로드 → 원하는 작업 폴더에 압축해제
