## 3. `useState` – 객체 상태 관리

**파일 경로:** `src/App.js`

```javascript
import React, { useState } from 'react';

function App() {
  // user: 사용자 정보 객체, setUser: 상태 변경 함수
  const [user, setUser] = useState({
    name: '',
    age: '',
  });

  // 입력값 변경 처리 함수
  const handleChange = (e) => {
    const { name, value } = e.target;
    // 기존 상태 복사 후 변경된 필드만 업데이트
    setUser((prevUser) => ({
      ...prevUser,
      [name]: value,
    }));
  };

  return (
    <div style={{ padding: 20 }}>
      <h1>사용자 정보</h1>
      <p>이름: {user.name}</p>
      <p>나이: {user.age}</p>

      <input
        type="text"
        name="name"
        placeholder="이름 입력"
        value={user.name}
        onChange={handleChange}
      />
      <input
        type="number"
        name="age"
        placeholder="나이 입력"
        value={user.age}
        onChange={handleChange}
      />

      <button onClick={() => setUser({ name: '', age: '' })}>
        초기화
      </button>
    </div>
  );
}

export default App;
```

---

### 코드 설명

1. **객체 형태로 상태 선언**

   ```javascript
   const [user, setUser] = useState({ name: '', age: '' });
   ```

   * `user` 객체 안에 여러 속성을 묶어 관리

2. **불변성 유지하며 업데이트**

   ```javascript
   setUser((prevUser) => ({
     ...prevUser,
     [name]: value,
   }));
   ```

   * 기존 객체를 복사(`...prevUser`)한 뒤 변경된 필드만 업데이트
   * 직접 `user.name = value`로 변경하면 React가 변경을 감지하지 못함

3. **초기화 버튼**

   ```javascript
   setUser({ name: '', age: '' });
   ```

   * 상태를 초기값으로 되돌림
