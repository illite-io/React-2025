## 53. `useClipboard` – 클립보드 복사

### 목적

* 텍스트나 데이터를 사용자의 클립보드에 복사하는 기능 제공
* 버튼 클릭 시 복사 → 토스트 알림/아이콘 변경 등의 UX 개선 가능

---

### 파일 경로: `src/hooks/useClipboard.js`

```javascript
import { useState } from 'react';

/**
 * useClipboard
 * @returns { copy, copied }
 *  - copy: 복사할 내용을 인자로 받아 클립보드에 저장하는 함수
 *  - copied: 직전에 복사 성공 여부 (boolean)
 */
export default function useClipboard() {
  const [copied, setCopied] = useState(false);

  const copy = async (text) => {
    if (!navigator?.clipboard) {
      console.warn('Clipboard API가 지원되지 않는 브라우저입니다.');
      return false;
    }
    try {
      await navigator.clipboard.writeText(text);
      setCopied(true);
      setTimeout(() => setCopied(false), 1500); // 1.5초 후 상태 초기화
      return true;
    } catch (err) {
      console.error('클립보드 복사 실패:', err);
      setCopied(false);
      return false;
    }
  };

  return { copy, copied };
}
```

---

### 파일 경로: `src/App.js`

```javascript
import React from 'react';
import useClipboard from './hooks/useClipboard';

export default function App() {
  const { copy, copied } = useClipboard();
  const textToCopy = 'https://nextstep.dev/hooks';

  return (
    <div style={{ padding: 20 }}>
      <h1>useClipboard – 클립보드 복사</h1>

      <p>복사할 URL: <code>{textToCopy}</code></p>

      <button
        onClick={() => copy(textToCopy)}
        style={{
          padding: '8px 14px',
          border: '1px solid #ddd',
          borderRadius: 6,
          background: copied ? '#10b981' : '#3b82f6',
          color: '#fff',
          cursor: 'pointer',
        }}
      >
        {copied ? '복사됨!' : '복사하기'}
      </button>
    </div>
  );
}
```

---

### 동작 설명

1. **Clipboard API 사용**

   * `navigator.clipboard.writeText()`로 문자열 복사
   * HTTPS 환경 또는 localhost에서만 동작

2. **상태 관리**

   * `copied` 상태로 버튼 텍스트/스타일 변경
   * 일정 시간 후 초기화로 UX 개선

3. **대체 방안**

   * 지원하지 않는 브라우저에서는 `document.execCommand('copy')` Fallback 가능
     (단, 최신 표준에서는 비권장)
